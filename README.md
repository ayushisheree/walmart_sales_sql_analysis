# Walmart Sales SQL Analysis

This project analyzes Walmart sales data using SQL to uncover business insights such as sales trends, product performance, and revenue analysis.

## 📊 Dataset
- Walmart sales dataset (CSV file)
- Contains store sales, temperature, fuel price, CPI, unemployment, and holiday flag details.

## 🛠 Tools Used
- MySQL
- SQL

## 📁 Project Structure

```bash
walmart_sales_sql_analysis/
│── data/
│   └── Walmart_Sales.csv
│
│── queries/
│   └── walmart_sales_queries_5.0.sql
│
└── README.md
```

## 🔍 Business Questions Solved
1. Which month had the highest sales?
2. What are the overall revenue trends?
3. How do holidays impact weekly sales?
4. Which stores performed the best?
5. What factors influence sales performance?

## 📈 Analysis Performed
- Monthly Sales Analysis
- Revenue Trends
- Store Performance Analysis
- Sales Insights Using SQL

## 🚀 How to Use
1. Import the dataset into MySQL.
2. Run the SQL query file.
3. Analyze the output for insights.
